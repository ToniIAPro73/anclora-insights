# Éxito sin compañía

Primer producto digital publicado bajo el sello Anclora Insights.

## Archivos maestros

- `source/`: documento editable del libro.
- `assets/cover/`: portada, contraportada y original conservado.
- `assets/marketing/`: mockup del ebook y fotografía de autor.
- `references/design/`: referencia visual usada para la landing.

## Publicación

### Shopify

- `publishing/shopify/ebook/`: PDF entregado como producto digital.
- `publishing/shopify/theme-packages/`: ZIP importable desde el administrador de Shopify.
- Tema editable: `shopify-themes/exito-sin-compania/`, repositorio Git independiente.

### Amazon KDP

- `publishing/amazon-kdp/ebook/`: EPUB e imagen de portada del ebook.
- `publishing/amazon-kdp/paperback/`: interior y cubierta envolvente para tapa blanda 6 × 9.
