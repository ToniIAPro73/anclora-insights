# Éxito sin compañía

Primer producto digital publicado bajo el sello Anclora Insights.
Disponible en español (edición original) e inglés (edición `en/`).

## Archivos maestros

- `source/`: documento editable del libro (`en/` para la edición en inglés).
- `assets/cover/`: portada, contraportada y original conservado (`en/` para la edición en inglés).
- `assets/marketing/`: mockup del ebook, banner para redes y fotografía de autor (`en/` para la edición en inglés).
- `references/design/`: referencia visual usada para la landing.

## Publicación

### Shopify

- `publishing/shopify/ebook/`: PDF entregado como producto digital (`en/Success_Without_Company_EN_Premium.pdf` para la edición en inglés).
- `publishing/shopify/theme-packages/`: ZIP importable desde el administrador de Shopify.
- Tema editable: `shopify-themes/exito-sin-compania/`, repositorio Git independiente.
- Estrategia de internacionalización: **Shopify Markets + Subcarpetas** — se usa la misma tienda y el mismo tema, sirviendo la edición en inglés bajo `anclora.com/en` mediante las herramientas nativas de traducción de Shopify. Centraliza inventario, clientes y pagos, y aprovecha la autoridad de dominio para el SEO internacional.

### Amazon KDP

- `publishing/amazon-kdp/ebook/`: EPUB e imagen de portada del ebook (`en/` para la edición en inglés, título comercial *Successful and Alone*).
- `publishing/amazon-kdp/paperback/`: interior y cubierta envolvente para tapa blanda 6 × 9 (`en/` para la edición en inglés).
